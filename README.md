# FansRockBar
**General Info
This repository holds a project for Fans Rock Bar's website - a rock club located in downtown Sofia. 
The website's layout is based on a w3schools responsive layout template and is customized to fit the client's needs. All assets used in the creation of the website were taken from the club's facebook page - https://www.facebook.com/rockbarfans/.
